# Severance MDR Software
 
The retro futuristic Macro Data Refinement software form Severance show, with emotion refinement as songs.

## Features

### Instructions Screen

A terminal-like instructions screen for the Macro Data Refinement software.

### File Explorer

A file explorer just as in the show.

## MDR Software

Micro data refinement ecosystem with numbers and refinement through happy, love, and sad songs.

## References

This UI is based on the Macro Data Refinement software from the TV show Severance and the repository: https://github.com/Imetomi/retro-futuristic-ui-design.git.


## Installation

This is now a complete React + TypeScript application using Vite with React Router for navigation.

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage For Devs

The application has shorcuts so the developers can navigate through screens quickly.

Screen 1: `Shift` + `1`
Screen 2: `Shift` + `2`
Screen 3: `Shift` + `3`
Screen 4: `Shift` + `4`
Screen 5: `Shift` + `5`
